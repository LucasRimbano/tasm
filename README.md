# Ejercicios de Assembly - Sistemas de Procesamiento de Datos

Este repositorio reúne ejercicios desarrollados en **Assembly** como parte de la materia **Sistemas de Procesamiento de Datos** de la **Universidad Nacional de San Martín (UNSAM)**.

## Propósito

El propósito de este repositorio es documentar mi práctica y evolución en el aprendizaje del lenguaje ensamblador, trabajando sobre ejercicios orientados a comprender el funcionamiento interno del procesador, la manipulación de registros, memoria e instrucciones de bajo nivel.

## Contenidos

Los ejercicios incluidos abarcan distintos temas, entre ellos:

- manejo de registros
- operaciones aritméticas y lógicas
- acceso y manipulación de memoria
- saltos condicionales e incondicionales
- estructuras de control
- resolución de consignas prácticas
